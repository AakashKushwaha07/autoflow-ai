# AutoFlow AI 🚀

AutoFlow AI is a self-healing enterprise workflow system that uses multi-agent architecture to automate, monitor, and recover workflows.

## 🔥 Features

- Workflow automation
- Multi-agent execution
- Failure detection
- Retry & recovery system
- Audit logs (explainability)
- React dashboard UI

## 🧠 Architecture

Frontend → React  
Backend → Spring Boot  
Database → H2  

Agents:
- Orchestrator
- Task Agents
- Monitoring Agent
- Recovery Agent

## 🧪 Demo Flow

1. Create workflow  
2. Execute workflow  
3. Observe failure  
4. Retry failed task  
5. Workflow recovers  

## 🚀 Future Scope

- LLM integration
- Slack/Email automation
- ML-based failure prediction

## ⚙️ How to Run

### Backend
cd backend
./mvnw.cmd spring-boot:run

### Frontend
cd frontend/autoflow-ui
npm install
npm run dev
